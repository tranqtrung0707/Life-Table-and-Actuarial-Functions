# Life-Table-and-Actuarial-Functions
An Excel Workbook containing a Life Table and a calculator for common Actuarial Functions.

For the Life Table, mortality follows Makeham's Law.
The default parameters are: $$A=0.0022, B=0.0000027, c=1.124$$ and interest rate $$i=5\\%$$. With those default parameters, the resulting table is the Standard Ultimate Life Table as given by the SOA for the FAM exam.

On the *Actuarial Functions* worksheet, the input parameters are $$x$$, the age at issue; $$n$$, the term; and $$m$$, the number of payments in a year. Given those parameters, the worksheet calculates the actuarial present value of  whole life, term and endowment insurance, and whole life and term annuity, for yearly payment, m-thly payment and continuous payment (using different approximations).
